# CSV Data Cleaner & Analyzer

Python automation project for cleaning, validating, and analyzing CSV data.

## Overview

This project helps process CSV files by:

- loading raw CSV data
- cleaning inconsistent or missing values
- standardizing columns and formats
- generating basic summary analysis
- exporting cleaned results for further use

## Features

- CSV file reading and parsing
- data cleaning and normalization
- missing value handling
- duplicate removal
- basic data analysis and reporting
- export of cleaned CSV output

## Requirements

- Python 3.10+
- pandas

## Installation

```bash
git clone <repository-url>
cd "Day 5 - CSV Data Cleaner & Analyzer"
pip install -r requirements.txt
```

If no `requirements.txt` is available, install dependencies manually:

```bash
pip install pandas
```

## Usage

Run the main script from the project directory:

```bash
python main.py
```

If your entry point has a different name, replace `main.py` with the correct file.

## Example Workflow

1. Place the source CSV file in the project folder.
2. Run the script.
3. Review cleaned data and analysis output.
4. Use the exported CSV for downstream tasks.
